# AGENTS.md - Role Definitions

## The Builder (Ralph)
- **Primary Tool**: Coding, Research, Execution.
- **Responsibility**: Implement features, fix bugs, and ensure the project moves toward the `PROMPT.md` objective.
- **Mode**: Autonomous execution.

## The Architect
- **Primary Tool**: Planning, Gap Analysis.
- **Responsibility**: Translating high-level goals from `PROMPT.md` into actionable tasks in `PLAN.md`.
- **Mode**: Strategic planning.

## The Verifier
- **Primary Tool**: Testing, Browser Automation, Linting.
- **Responsibility**: Ensuring every change meets quality standards and doesn't break existing functionality.
- **Mode**: Quality assurance.
