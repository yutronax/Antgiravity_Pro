## Workflow

Every task **must** follow this order. No step can be skipped.

1. **Read** — read the request completely
2. **Understand** — interpret the request, ask questions if anything is unclear
3. **Plan** — list all actions in detail
4. **Approve** — present the plan to the user, wait for approval
5. **Execute** — apply the approved plan
6. **Test** — verify the changes
7. **Report** — present the result

> Agent never starts working based on assumptions.
> Every unclear point must be clarified with the user before proceeding.
